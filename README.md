### Framez
Framez is a mobile social platform built with React Native, Expo, and Supabase that allows users to create accounts, post text and images, and interact in a real-time feed.

## Features

- User authentication (signup, login, logout) using Supabase Auth

- Profile management

- Create posts with optional images

- Real-time feed updates using Supabase Realtime

- View posts with author information

- Delete posts


### Tech Stack

- Frontend: React Native, Expo

- Backend & Database: Supabase (PostgreSQL + Realtime + Storage)

- Routing & Navigation: Expo Router

- State Management: React Context (AuthContext, PostsContext)

- Storage: AsyncStorage (for persisting session data)
